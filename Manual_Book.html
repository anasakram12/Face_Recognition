0 How to Open Terminal in Linux (Ubuntu)

If you're new to Linux, here's how to open the terminal window where you'll enter commands:

Method 1: Keyboard Shortcut

⌨️ Press these keys together:
Keyboard Shortcut
Ctrl + Alt + T

This opens a new terminal window.

Method 2: Using the Application Menu

🖱️ GUI Method:
  1. Click the "Activities" or "Show Applications" button on your desktop (usually bottom-left or top-left).
  2. Type "Terminal" in the search box.
  3. Click the Terminal icon that appears.

4 Stopping the System

You can stop the running scripts by either:

⌨️ Method 1: Graceful Shutdown

Pressing Ctrl + C inside each terminal window running a process to gracefully stop it.

⚠️ OR
🖱️ Method 2: Force Close

Simply close the terminal window by clicking the "X" button at the top corner of the terminal window. This will terminate the running script immediately.

Alternative: Kill Processes by Name

If needed, you can kill all related processes using:

Force Kill Commands
pkill -f Mod_Sentinel_server_Det.pyc pkill -f flask_server.py pkill -f Database_Web.py pkill -f run_all.sh
